# Knowledge-Base

**Knowledge-Base** is a central platform for organizing, managing, and sharing various specialized knowledge bases and AI-powered support tools. It supports decision-making across multiple domains, ensuring reliable access to key information for all stakeholders.

---

## Repository Structure & Contents

### 1. Avocado_Knowledge_base
This section provides an expert system for managing avocado crop health.
- **Purpose:** Supports avocado farmers and researchers in identifying pests and diseases and recommending appropriate treatments.
- **Details:** Uses advanced logic programming with a user-friendly interface to offer:
  - Pest and disease identification and management.
  - In-depth filtering and explanation of control options, including organic and biological solutions.
  - Fast responses based on logical rules, not just lookups.
  - See the [detailed README here](https://github.com/Lord-LLM/Knowledge-Base/blob/main/Avocado_Knowledge_base/README.md).

---

### 2. Deans_FAQ_system
A smart FAQ knowledge base for university Deans’ offices and administration.
- **Purpose:** Allows staff and students to get fast, consistent answers to common questions about academic policies, registration, finances, graduation, and other university processes.
- **Details:** Combines logic-driven search with a database of FAQs and synonym support to provide relevant answers and suggestions.
  - The FAQ logic is implemented with correctness checks to ensure accurate and fair information.

---

### 3. Knowledge_Base
A collection of general-purpose logic modules and visualization tools for knowledge management.
- **Purpose:** Provides logic-driven components supporting the other knowledge bases. Used mainly for structuring data, querying facts, and visualizing results.
- **Details:** 
  - Includes code for logic programming queries, simple data visualization utilities, and requirements for running the modules.

---

### 4. Smart_Greenhouse_Decision_Support_System
A rule-based simulation of intelligent decision-making for greenhouse operations in challenging environments.
- **Purpose:** Models real-time decisions for a greenhouse in semi-arid conditions, focusing on sustainability and risk management.
- **Details:**
  - Simulates actions such as automatic watering, shading, alerting, and escalation based on fuzzy logic and AI rules.
  - Helps researchers and engineers study and optimize greenhouse management.
  - See the [system README here](https://github.com/Lord-LLM/Knowledge-Base/blob/main/Smart_Greenhouse_Decision_Support_System/README.md).
